Build a prototype

Use AI agents with specialist skills to build prototypes with the GOV.UK prototype kit. This workflow gives you a repeatable process with clearly defined review steps. You'll need to tailor this workflow to suit your team, and improve it over time.

Who this is for

Use this workflow if you:

  • build prototypes regularly
  • want a repeatable process that improves over time
  • have designs ready to turn into working code

Do not use this workflow if you:

  • need well tested, production-ready code
  • want help creating designs
  • have a one-off need. This is designed for teams who prototype often
  • Acknowledge: This is a work in progress. For AI help beyond this workflow, contact us in #ask-ace.

What you'll need

  • A design, such as Figma files, screen descriptions, or journey maps
  • Someone comfortable with Git, Node, and the CDP deployment process
  • An AI coding tool, such as Cursor, Claude Code, or GitHub Copilot
